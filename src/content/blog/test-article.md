## はじめに

TypeScriptは静的型付けを持つJavaScriptのスーパーセットです。この記事では基本的な型システムについて説明します。

## セクション1

型注釈を使うことで、変数や関数の型を明示的に指定できます。これにより、開発時にエラーを早期に発見できるようになります。

```typescript
const message: string = "Hello, TypeScript!";
const count: number = 42;
```

## まとめ

TypeScriptの型システムを活用することで、より安全で保守性の高いコードを書くことができます。
